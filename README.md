# Multi-Agent CSV Analyst

A Streamlit app that uses **LangGraph** and **Google Gemini** to answer questions about your CSV files. Upload any dataset, ask in plain English, and get a Markdown report—without sending full row data to the model (only schema + a small sample).

## Status

Project setup in progress. More docs and setup steps will be added soon.
